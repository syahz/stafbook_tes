
# Run like a Copy Laravel.



## Installation

1.  Clone the repo

2.  Update Composer
```bash
    composer update
    cd stafbook_tes
```

3. Copy .env example jadi .env
```bash
    cp .env.example .env
```

4. Generate Key
```bash
    php artisan generate key
```

5.  Run migrate for laravel
```bash
    php artisan migrate
```

4.  Run
```bash
    php artisan ser
  ```
